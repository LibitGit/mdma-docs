# Okno Aktywnych Ustawień

#### Ustawienia tekstu
<p><img src="images/text-settings.png" class="right" style="padding-left: 10px;" alt="Ustawienia tekstu" /></p>

* Wybór rozmiaru tekstu.
* Wybór koloru tekstu.
